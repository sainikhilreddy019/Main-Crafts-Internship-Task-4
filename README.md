# Task 4: Interactive Sales Dashboard using Power BI

## Objective
Create an interactive business dashboard using Power BI to analyze sales performance and provide business insights.

## Tools Used
- Power BI Desktop
- Excel

## Dataset
- Sales Data
- Customers Data
- Products Data

## Dashboard Features
- Total Sales KPI Card
- Sales by Category
- Sales by Region
- Sales by Segment
- Sales by Sub-Category
- Interactive Filters (Region, Category, Segment)

## Key Insights
- Total Sales exceeded 2.26M.
- Technology category generated the highest sales.
- West region recorded the highest sales performance.
- Consumer segment contributed the largest share of revenue.
- Sales trends varied across product sub-categories.
